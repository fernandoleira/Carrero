# Carrero

Robotic vehicle that can be controlled remotly with a Raspberry Pi.

## Getting Started

```
git clone https://github.com/fernandoleira/Carrero
cd Carrero
```

## Components

* Raspberry PI 2
* Portable Battery
* 9V Battery
* L298N Steeper Motor Driver
* EMO Smart Robot Car Chassis Kit
